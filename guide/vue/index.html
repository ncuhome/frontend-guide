# vue/eqeqeq

必须使用 === 或 !==，禁止使用 == 或 !=

// bad
<template>
    <div :attr="foo == 1" />
</template>

// good
<template>
    <div :attr="foo === 1" />
</template>

# vue/valid-v-bind

v-bind 指令必须合法

// bad

    <div>
        <div v-bind></div>
        <div :aaa></div>
        <div v-bind:aaa.bbb="foo"></div>
    </div>
</template>

// good
<template>
    <div>
        <div v-bind="foo"></div>
        <div v-bind:aaa="foo"></div>
        <div :aaa="foo"></div>
        <div :aaa.prop="foo"></div>
    </div>
</template>
